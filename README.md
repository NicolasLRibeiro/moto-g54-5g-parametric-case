# Moto G54 5G – Parametric Smartphone Case

Parametric smartphone case designed in **OpenSCAD**, developed for 3D printing and customization.

## Overview

This project contains a fully parametric protective case for the Moto G54 5G.  
The design allows dimension adjustments and printing optimization directly through the `.scad` file.

## Preview

![Case Preview](images/PRINT_GERAL.jpeg)

## Features

- Parametric design
- Camera cutout
- USB-C port opening
- Speaker and AUX openings
- Front lip protection
- Optimized for FDM 3D printing

## Usage

1. Open the `.scad` file in OpenSCAD  
2. Render the model (F6)  
3. Export as STL  
4. Slice and print
